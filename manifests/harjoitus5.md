#Harjoitus 5 raportti

h5. Tee moduli git-varastoon ja kirjoita raportti sinne MarkDownilla.

Eli tein tehtävän läppärilläni ja käytin jo olemassa olevaa koodia, joka löytyi /etc/puppet/modules/testi alta.
 Aluksi kirjauduin githubiin ja otin sieltä ohjeet miten olemassa oleva tiedosto saadaan tallennettua sinne.
Seuraavaksi asensin gitin kannettavalleni ja käytn komennot git init, git add., ja git commit. annoin tselleni nimen ja kirjoitin s-posti osoitteeni. 
seuraavaksi committasin uudestaan ja laitoin push komenon " git remote add" ja netti osoite perään. Ja vielä git push origin master
